$\underline{主题切换}$ 即通过点击某个组件来切换背景与文字的颜色等 CSS 属性。这要求我们的组件能够控制某个”环境变量”, 且index.css中的 CSS 应随这个”环境变量”而改变。

要实现这个需求,我们需要解决以下几个问题：

  1. 如何定义和管理这个”环境变量”？
  2. 组件如何控制这个变量？
  3. CSS 如何响应变量的变化？

这些问题的解决方案涉及到几个重要的基础概念,让我们逐一了解。

Context 提供了一种在组件树中共享数据的方式,无需手动在每一层传递 props。
